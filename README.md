# Солодка майстерня

Сайт-каталог десертів та випічки з сучасним адаптивним інтерфейсом.

## Про проєкт

Солодка майстерня — це вебзастосунок для кондитерської або онлайн-вітрини
десертів.

Користувач може:

- переглядати список десертів;
- фільтрувати десерти за категоріями;
- переглядати картки товарів;
- отримувати базову інформацію про десерти та їхню вартість.

## Технології

- HTML5
- CSS3
- JavaScript (ES6+)
- Vite
- REST API
- Git / GitHub

## Run Locally

Clone the project

```bash
  git clone https://github.com/VadymHromyk/project-codex133
```

Go to the project directory

```bash
  cd project-codex133
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
